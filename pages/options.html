AI analysis

Add an API key from any supported provider and ShadowShield's AI layer activates: suspicious pages are automatically double-checked and can be escalated to blocked. The key is stored only on this device — never synced to your Google account or any other machine — and is only ever sent to the provider you choose. Analysis requests contain the page URL, title, and a short text sample — never anything you've typed.

Works with any service that speaks the OpenAI chat-completions API — including a local Ollama server (base URL http://localhost:11434/v1, any key text).
